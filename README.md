# Competition Projects

## About

단순히 모델을 적용하는 것보다, 데이터가 가진 특성을 문제에 맞게 구조화하고 모델이 이를 잘 반영하도록 설계하는 데 초점을 두었습니다.
각 프로젝트를 통해 데이터의 의미를 파악하고, 성능 개선으로 연결하는 실험 중심의 접근을 경험했습니다.
