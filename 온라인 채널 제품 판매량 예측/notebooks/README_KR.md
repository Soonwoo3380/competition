## 📖 프로젝트 배경

제공된 데이터는 **제품 코드만 존재**하여 모델이 제품의 실제 특성을
학습하기 어려웠습니다.

단순히 판매량 패턴만 학습하는 것이 아니라 **도메인 지식을 활용한 Feature
Engineering**을 통해 모델이 학습할 수 있는 정보를 확장하는 것을 목표로
하였습니다.

------------------------------------------------------------------------

## 🎯 문제 정의

Baseline은 Sliding Window 기반 판매량만 학습했습니다.

하지만 실제 판매량은 다음과 같은 요소의 영향을 받습니다.

-   제품 카테고리
-   브랜드
-   재구매 주기
-   함께 구매되는 제품
-   요일
-   공휴일

> **제품 코드 이상의 정보를 모델에 학습시킬 수 없을까?**
